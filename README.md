# 🎯 Object Detection Using YOLO

A deep learning-based **Object Detection** project that detects and identifies multiple objects in images using the **YOLO (You Only Look Once)** algorithm. This project is implemented in Python using a Jupyter Notebook and demonstrates fast, accurate, and real-time object detection.

---

## 📌 Overview

Object Detection is a computer vision technique that identifies the location and class of objects within an image. This project uses the YOLO model to detect objects with high accuracy and display bounding boxes along with class labels and confidence scores.

---

## ✨ Features

- 🔍 Detect multiple objects in a single image
- 📦 Draw bounding boxes around detected objects
- 🏷️ Display object labels and confidence scores
- ⚡ Fast and efficient YOLO-based detection
- 📒 Easy-to-use Jupyter Notebook implementation

## 📦 Installation

### 1. Clone the repository

```bash
git clone https://github.com/yourusername/object-detection.git
cd object-detection
```

### 2. Install dependencies

```bash
pip install opencv-python numpy matplotlib ultralytics
```

---

## ▶️ Usage

1. Open the notebook:

```bash
jupyter notebook objectdetection.ipynb
```

2. Run all cells.

3. Upload or specify an input image.

4. The model detects objects and displays:

- Bounding boxes
- Object names
- Confidence scores

## 📚 How It Works

1. Load the YOLO model.
2. Read the input image.
3. Perform object detection.
4. Generate bounding boxes.
5. Display class labels and confidence scores.
6. Show the final output image.
